# Mahjong Soul Link Changer (작혼 링크 변환기)

이 크롬 확장 프로그램은 사용자의 설정에 따라 작혼 게임 URL을 글로벌, 한국, 일본 서버 간에 자동으로 변환해줍니다.

## 기능

- **다중 서버 지원**: 글로벌(`mahjongsoul.game.yo-star.com`), 한국(`mahjongsoul.game.yo-star.com/kr`), 일본(`game.mahjongsoul.com`) 서버 간을 쉽게 전환할 수 있습니다.
- **자동 변환**: `room` (친선전) 및 `paipu` (게임 로그) 링크를 감지하여 선택한 서버로 자동 리디렉션합니다.
- **사용자 친화적 인터페이스**: 간편한 팝업 인터페이스를 통해 목표 서버를 선택할 수 있습니다.
- **다국어 지원**: 영어, 한국어, 일본어를 지원합니다.

## 설치 방법

1. 이 저장소를 복제하거나 다운로드합니다.
2. 크롬을 열고 `chrome://extensions`로 이동합니다.
3. 우측 상단의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 클릭합니다.
5. 이 프로젝트의 `src` 디렉토리를 선택합니다.

## 권한 설명

이 확장 프로그램이 올바르게 작동하려면 다음 권한이 필요합니다:

- **`tabs`**: 현재 탭의 URL에 접근하기 위해 필요합니다. 이를 통해 작혼 링크 방문을 감지하고 쿼리 파라미터(`room` 또는 `paipu`)를 읽어 리디렉션을 수행할 수 있습니다.
- **`storage`**: 선호하는 대상 서버(글로벌, 한국, 일본) 설정을 저장하기 위해 필요합니다. 브라우저를 껐다 켜도 설정이 유지됩니다.
- **`webNavigation`**: 탐색 이벤트를 더 확실하게 감지하기 위해 필요합니다. 페이지 전체 로드 없이 URL이 변경되는 경우(SPA 등)나 탐색을 조기에 가로채는 데 도움이 됩니다.

## 사용법

1. 크롬 툴바에서 확장 프로그램 아이콘을 클릭합니다.
2. 원하는 서버(Global, KR, JP)를 선택합니다.
3. 작혼 링크(예: `?room=...`)를 방문하면 선택한 서버로 자동 리디렉션됩니다.
